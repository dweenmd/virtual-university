University of Dhaka Affiliated Engineering College Syllabus (Session 2019-20). Total Credits: 160.50

1st Year, 1st Semester

Semester I

📚 6 Theory 🧪 3 Lab 🎯 20.50 Total Credits
CSE-1101

Fundamentals of Computers and Computing

2.0 CreditsPrereq: None

Course Content:Introduction to computer systems, hardware components, history of computing, computing environments, binary system representation, numbering systems, software architectures, algorithms and flowcharts concept, basic system logic and application layers.

CSE-1102

Discrete Mathematics

3.0 CreditsPrereq: None

Course Content:Set theory, mathematical logic, propositional and predicate calculus, relations, functions, graph theory, trees, algebraic systems, groups, rings, combinatorics, recurrence relations, and permutations.

EEE-1103

Electrical Circuits

3.0 CreditsPrereq: None

Course Content:DC circuit analysis, Ohm's Law, Kirchhoff's current and voltage laws, mesh and nodal analysis, network theorems (Thevenin, Norton, Superposition), AC circuit basics, phasors, RLC circuits, complex power.

CHE-1104

Chemistry

3.0 CreditsPrereq: None

Course Content:Atomic structure, periodic classification, chemical bonding, thermodynamics, electrochemistry, chemical kinetics, solutions, phase rules, and introductory environmental chemistry.

MATH-1105

Differential and Integral Calculus

3.0 CreditsPrereq: None

Course Content:Functions, limits, continuity, differentiation of algebraic and transcendental functions, successive differentiation, Leibnitz's theorem, Mean Value theorems, Taylor and Maclaurin expansions, integration techniques, definite integrals, areas under curves.

SS-1106

Government and Public Administration

2.0 CreditsPrereq: None

Course Content:Basic concepts of governance, state apparatus, constitution of Bangladesh, public policy formulation, administrative systems, and local government frameworks.

1st Year, 2nd Semester

Semester II

📚 5 Theory 🧪 4 Lab 🎯 21.50 Total Credits
CSE-1201

Fundamentals of Programming

3.0 CreditsPrereq: CSE-1101, CSE-1102

Course Content:Basic I/O, Data Types, Conditionals, Loops (while, for, do-while), Formatted I/O, Nested Loops. Functions (Call by Value/Reference, Recursion). Arrays (1D, 2D, Multidimensional). Strings operations without library functions, structures, pointers architecture, dynamic memory allocation (malloc, calloc, realloc, free), file management operations, preprocessors, and macros.

CSE-1202

Digital Logic Design

3.0 CreditsPrereq: None

Course Content:Number systems, logic gates, Boolean algebra, K-Maps, Minimization schemas. Latches, Flip-Flops (SR, JK, D, T), Counters (Asynchronous, Synchronous), Shift registers, decoders, encoders, multiplexers, demultiplexers, RAM/ROM architectures, and adder/subtractor arithmetic circuits.

PHY-1203

Physics

3.0 CreditsPrereq: None

Course Content:Thermodynamics laws, entropy, Carnot cycle. Structure of matter (crystallography, Miller indices, Bragg's law, band theory). Waves & Oscillations (SHM, damped/forced oscillations, acoustics, resonance). Physical optics (interference, diffraction, polarization, gratings).

MATH-1204

Methods of Integration, Differential Equations and Series

3.0 CreditsPrereq: MATH-1105

Course Content:Integration by parts, trigonometric substitutions, partial fractions. Modeling first order differential equations, separable/linear equations. Polar coordinates. Infinite sequences and convergence tests, Power series, Taylor and Maclaurin polynomials.

ENG-1205

Developing English Language Skills

2.0 CreditsPrereq: None

Course Content:English phonetics, vocabulary enrichment techniques, structural mechanics, reading comprehension strategies, essay and discourse development, business correspondence, report structures, and technical writing foundations.

Semester III

📚 5 Theory 🧪 3 Lab 🎯 21.00 Total Credits
CSE-2101

Data Structures and Algorithms-I

3.0 CreditsPrereq: CSE-1201

Course Content:Arrays, Linked Lists (Singly, Doubly, Circular), Stacks and Queues applications. Searching (Linear, Binary), Basic Sorting (Bubble, Insertion, Selection, Merge, Quick). Trees basics (BST operations), Hash tables, recursion analysis and Time Complexity (Big-O notation).

CSE-2102

Object Oriented Programming-I

3.0 CreditsPrereq: CSE-1201

Course Content:Introduction to OOP paradigms, Classes, Objects, Abstraction, Encapsulation, Inheritance, Polymorphism. Method overloading and overriding, constructors, access modifiers, interfaces, exception handling, and file I/O operations inside object contexts (C++/Java syntax basics).

EEE-2103

Electronic Devices and Circuits

3.0 CreditsPrereq: EEE-1103

Course Content:Semiconductor diodes, Zener diodes, Rectifier circuits. Bipolar Junction Transistors (BJT) characteristics, biasing networks, amplifiers. Field Effect Transistors (FET/MOSFET) architectures, operational amplifiers (Op-Amps) and basic applications.

MATH-2104

Linear Algebra

3.0 CreditsPrereq: None

Course Content:Matrices and determinants, systems of linear equations, Gaussian elimination. Vector spaces, linear independence, basis and dimension. Linear transformations, Eigenvalues, Eigenvectors, and diagonalization maps.

MATH-2105

Coordinate Geometry and Vector Analysis

3.0 CreditsPrereq: None

Course Content:2D and 3D coordinate geometry transformations, straight lines, planes, spheres. Vector calculus, dot/cross products, gradient, divergence, curl theorems (Green's, Gauss, Stokes equations).

Semester IV

📚 5 Theory 🧪 3 Lab 🎯 19.50 Total Credits
CSE-2201

Data Structures and Algorithms-II

3.0 CreditsPrereq: CSE-2101

Course Content:Advanced graph strategies, BFS, DFS, shortest paths (Dijkstra, Bellman-Ford, Floyd-Warshall). Minimum Spanning Trees (Prim, Kruskal). Greedy algorithms, Dynamic Programming (LCS, Knapsack), NP-Completeness introductions.

CSE-2202

Database Management Systems-I

3.0 CreditsPrereq: None

Course Content:Database concepts, ER-modeling, relational models. Relational algebra. SQL schemas (DDL, DML, relational assertions). Normalization techniques (1NF, 2NF, 3NF, BCNF). File organization, basic indexing architectures.

CSE-2203

Computer Architecture

3.0 CreditsPrereq: CSE-1202

Course Content:Design components of Von Neumann machine, CPU units, pipelining architectures. ALU designs, arithmetic registers, control unit operations (hardwired, microprogrammed). Memory hierarchy, cache configurations, virtual memory, and I/O setups.

MATH-2204

Complex Variable and Statistical Analysis

3.0 CreditsPrereq: None

Course Content:Complex functions, Cauchy-Riemann equations, residue integration. Probability definitions, random variables, distributions (Binomial, Poisson, Normal). Hypothesis metrics, data regressions.

ECON-2205

Economics

2.0 CreditsPrereq: None

Course Content:Micro & Macroeconomics foundations. Laws of supply and demand, cost optimization analyses, national income audits, inflation indexes, monetary metrics, and project budget balancing frameworks.

Semester V

📚 5 Theory 🧪 3 Lab 🎯 19.50 Total Credits
CSE-3101

Software Engineering

3.0 CreditsPrereq: None

Course Content:Software lifecycle paradigms (Waterfall, Agile, Scrum). Software Requirement Specification (SRS) architecture, UML design maps, architectural patternings, debugging paradigms, validation testing matrices, and project scaling analytics.

CSE-3102

Data Communication

3.0 CreditsPrereq: None

Course Content:Signals and signal properties, Fourier transforms. Transmission media, analog-to-digital conversions, modulation mappings (ASK, FSK, PSK, QAM). Multiplexing pipelines (FDM, TDM). Error corrections (CRC protocols).

CSE-3103

Operating Systems

3.0 CreditsPrereq: CSE-2203

Course Content:OS services, process threads, scheduling matrices (FIFO, Round Robin, Priority metrics). Critical sections, semaphores, deadlocks mitigation (Banker's algorithm). Paging, segmentation hierarchies, file management strategies.

CSE-3104

Theory of Computation

3.0 CreditsPrereq: CSE-1102

Course Content:Finite Automata (DFA, NFA definitions), Regular expressions, pumping lemmas. Context-free grammars, Pushdown Automata (PDA models). Turing Machine computations, halting problems, undecidability metrics.

CSE-3105

Microprocessors and Microcontrollers

3.0 CreditsPrereq: CSE-2203

Course Content:Intel 8086 microarchitecture architecture, register sets, segmentation logic, addressing controls. Pin layout systems, memory bank interfaces. Programmable peripheral controls (8255, 8254). 8051 microcontroller cores, register models.

Semester VI

📚 5 Theory 🧪 3 Lab 🎯 19.50 Total Credits
CSE-3201

Compiler Design

3.0 CreditsPrereq: CSE-3104

Course Content:Lexical processing implementations, Tokenizers, Lex/Flex tools. Parsing engine design (LL, LR parsers, Yacc/Bison setups). Syntax Directed Translations, intermediate syntax representations, DAG graphs, code emission optimization algorithms.

CSE-3202

Computer Networks

3.0 CreditsPrereq: CSE-3102

Course Content:ISO/OSI & TCP/IP stack layers. Data link frames, sliding window controls, routing engines (RIP, OSPF, BGP protocols). Transport sockets (TCP/UDP mechanisms), congestion control loop structures, and application level controls (DNS, HTTP architectures).

CSE-3203

Object Oriented Programming-II

3.0 CreditsPrereq: CSE-2102

Course Content:Advanced Java/C#, generic class data configurations, multithreading control models, runtime reflection abstractions. GUI toolkit event patterns, network programming adapters (Socket API), database binding engines (JDBC/Hibernate mappings).

CSE-3204

Database Management Systems-II

3.0 CreditsPrereq: CSE-2202

Course Content:Query compiler metrics, relational cost transformations. Transaction ACID logs, serialization checking maps. Deadlock avoidance logic. Distributed database architectures, NoSQL models, warehousing pipelines.

CSE-3205

Technical Writing and Presentation

1.5 CreditsPrereq: None

Course Content:Research documentation frameworks, LaTeX layout structuring, IEEE submission style standards, data visualizations, and project poster design methods.
